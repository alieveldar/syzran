Эта папка для баннерной системы
Так же необходимы папки:
- statistic (777) - папка с файлами статистики
- domains (777) - папка с файлами-списками текущих баннеров по доменам
- files (777) - папка с файлами баннеров
- cache (777) - папка с таблицами баннеров


Файлы баннерной системы:
- getBanner.php - получение списка баннеров по текущему домену
- showBanner.php - счетчик отображений баннера
- clickBanner.php - счетчик кликов по баннеру, а так же редирект на нужную страницу
- initBanner.php - создает папки, чистит мусор, переподготавливает списки баннеров
- preBanner.php - показ отдельного баннера (предварительный просмотр для утверждения)
