Эта папка содержит стандартные оформления элементов сайта, так же здесь хранятся стили и файлы стандартных модулей.
Для добавления своего шаблона создайте папку в директории "template", внутри разместите файл стилей css с названием как и новая папка.
Например: 
Cоздаем папку "newdesign", где будет новый дизайн
Внутри папки "newdesign" создаем "newdesign.css", где описываем все стили.
Внутри папки "newdesign" создаем "newdesign.html", где содержится верстка сайта.
Внутри папки "newdesign" создаем "newdesign.js", где скрипт дизайна.
